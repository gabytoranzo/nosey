# Neon POV Quest

A character jumping mission built with plain HTML, CSS, and JavaScript. Choose Dad, G, or Pau, collect every crystal, press both switches, and reach the exit gate.

Source link shown in game: https://github.com/gabytoranzo/nosey/tree/main/neon-pov-quest

## Play

Open `index.html` in a browser, or run a local server:

```sh
python3 -m http.server 5175
```

Then visit `http://localhost:5175`.

## Controls

- Choose Dad, G, or Pau from the start screen to begin
- Arrow keys or A/D: move left and right
- Up, W, or Space: jump
- R or the R button: return to the last safe platform
- P or Escape: pause
- Touch: swipe left/right to nudge, swipe up to jump
