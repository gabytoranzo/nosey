# Neon Drift

A Pac-Man-style browser maze game built with plain HTML, CSS, and JavaScript.

## Play

Open `index.html` in a browser, or run a local server:

```sh
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.

## Controls

- Choose Dad, G, or Pau from the start screen to begin.
- Arrow keys or WASD: move through the maze
- Power pellets make ghosts vulnerable for a few seconds
- P or Escape: pause
- Touch: swipe to change direction
