# Neon Drift

A fast little browser game built with plain HTML, CSS, and JavaScript.

## Play

Open `index.html` in a browser, or run a local server:

```sh
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.

## Controls

- Left / Right arrows or A / D: switch lanes
- Space, W, or Up arrow: fire pulse when charge reaches 100%
- P or Escape: pause
- Touch: swipe to move, tap to pulse
